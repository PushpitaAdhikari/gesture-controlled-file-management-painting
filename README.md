# Gesture-Controlled File Management and Painting Application using Human Computer Interaction

## Overview
This project is a Human-Computer Interaction (HCI) based application that enables users to control file management operations and digital painting functions through hand gestures captured via a webcam.

## Features
- Gesture-based file navigation
- Open, close, select, and manage files using hand gestures
- Virtual painting/drawing interface
- Real-time hand tracking
- Touchless interaction system

## Technologies Used
- Python
- OpenCV
- MediaPipe
- NumPy
- PyAutoGUI
- Tkinter / Pygame

## Project Objective
To create an interactive and touch-free system that improves accessibility and user experience through gesture recognition.

## Modules
- Hand Detection and Tracking
- Gesture Recognition Engine
- File Management Controls
- Virtual Painting Interface
- User Interaction Module
